# わんにゃんメモリー - セットアップガイド

## フォルダ構成
```
wannyan-memory/
├── index.html        メイン画面
├── style.css         スタイル
├── app.js            アプリロジック
├── manifest.json     PWA設定
├── sw.js             Service Worker（オフライン対応）
├── icons/
│   ├── icon-192.png  ← ご自身のアイコン画像を入れてください
│   └── icon-512.png  ← ご自身のアイコン画像を入れてください
└── README.md
```

## セットアップ手順

### 1. アイコン画像を用意する
`icons/` フォルダ内に以下の2種類の画像を入れてください：
- `icon-192.png`：192×192ピクセルのPNG画像
- `icon-512.png`：512×512ピクセルのPNG画像

### 2. サーバーへのアップロード（HTTPS必須）
PWAはHTTPSが必要です。以下のいずれかを使ってください：

#### 方法A：GitHub Pages（無料・簡単）
1. GitHubアカウントを作成
2. 新しいリポジトリを作成（例：wannyan-memory）
3. フォルダ内のファイルをすべてアップロード
4. Settings → Pages → Branch: main → Save
5. `https://あなたのユーザー名.github.io/wannyan-memory/` でアクセス可能

#### 方法B：レンタルサーバー
HTTPS対応のレンタルサーバーにフォルダごとアップロードするだけでOK。

#### 方法C：Netlify（無料・簡単）
1. https://netlify.com にアクセス
2. フォルダをドラッグ＆ドロップするだけ
3. 自動でHTTPS URLが発行される

### 3. iPhoneのホーム画面に追加
1. iPhoneのSafariでアプリのURLを開く
2. 下部の「共有」ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 名前を確認して「追加」をタップ

これでアイコンがホーム画面に追加され、アプリのように使えます！

## 機能一覧
- 🐕 犬 / 🐈 猫 の切り替え
- 名前順・記録日順の並び替え
- ひらがな・カタカナ対応の名前検索
- 写真・名前・年齢・生年月日の記録
- 生年月日入力で年齢自動計算
- 問題ごとのフォルダ管理（アコーディオン）
- 全体メモ
- オフライン対応（Service Worker）
- データはブラウザ内（localStorage）に保存

## 注意事項
- データはブラウザのlocalStorageに保存されます
- iPhoneを初期化するとデータが消えます
- 写真を登録すると容量が大きくなります（目安：1枚あたり数百KB〜数MB）
- SafariのプライベートブラウジングではService Workerが動作しません
